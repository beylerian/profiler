### Author Profiler

Detects simple latent attributes in English text samples

Uses Apache OpenNLP for classification

Uses Apache Tika for document parsing

- Female

#### Age range detection

- Quinary :  [18-24],  [25-34], [35-49], [50-64], [65-xx]

- Australia

- New zealand
