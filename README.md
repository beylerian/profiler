Uses Apache OpenNLP for classification

Uses Apache Tika for document parsing

- Female

#### Age range detection

- young [~20s]
- mature [20s+]

#### Variety profiling

- Great britain
- Australia
- Ireland
- New zealand

- extroverted
- stable/neurotic
- agreeable
- conscientious
- open
